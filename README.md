## Usage

```sh
npm install @motionpicture/express-middleware
```

```js
var middlewares = require('@motionpicture/express-middleware');
```

### Debug

DEBUG環境変数をセットすることでリクエストのIN,OUTをデバッグできます。

```shell
set DEBUG=express-middleware:*
```
